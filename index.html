What to do

Mobile Web Frameworks


  • JQuery Mobile
  • Sencha Touch
  • mGWT
  • backbone.js

JQuery Mobile


  • nice UI components
  • odd page loading concept
  • dialogs as a single page
  • 144KB

Sencha Touch


  • cross-plattform UI components
  • full mobile framework
  • configuration over programming
  • heavy DOM
  • customization difficult
  • 500KB

mGWT


  • cross-plattform, multi-channel approach
  • few UI components
  • plattform optimized code
  • slow iteration cycles

backbone.js


  • lightweight
  • flexible
  • just MVC, no components

How to do


  • shallow DOM
  • CSS3 where possible
  • small resource footprint
  • modularized, testable
  • fast iteration cycles
